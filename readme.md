## 项目名称

> hb-utils 前端 utils 库
> base 基层
> functions 函数
> plugins 插件：encrpt echarts

## 运行条件

> 列出运行该项目所必须的条件和相关依赖

- rollup
- rollup-plugin-terser
- @babel/core
- rollup-plugin-babel
- rollup-plugin-cleanup"

## 运行说明

> 项目中运用到的公共方法[utils 文件夹下的]

- 切换到公司外部的 npm 镜像
- npm install hb-utils -S
- 方案一:import { A(方法名) } from 'hb-utils'
- 卸载包 npm uninstall hb-utils
